# Blogchain

### A decentralized blogging platform

# Technologies
- Solidity
- Hardhat
- Polygon
- IPFS
- Moralis
- React

## Steps to run in the local environment
- Fork and clone this repository
- Install dependencies using ``` npm ``` or ``` yarn ```
- Run ``` yarn start ```

[Blogchain Deployment Link](https://ayush-blogchain.vercel.app)
