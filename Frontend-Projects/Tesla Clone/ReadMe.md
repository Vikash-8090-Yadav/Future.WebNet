# Tesla Clone

A tesla clone created using ReactJs and Tailwind CSS.

Link: [Tesla Clone](https://tesla-clone-tau-six.vercel.app/)

## Run Locally

Install dependencies

```bash
npm install --legacy-peer-deps
```

Command to start 

```bash
npm run dev
```
